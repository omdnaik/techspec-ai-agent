# US-013 – Artifact Metadata Builder Framework

## Priority

P0 (Critical Path)

## Goal

Implement the Artifact Metadata Builder responsible for creating one runtime Artifact Metadata object for every artifact defined in the validated manifest.

Artifact Metadata is the primary runtime object consumed by all pipeline lifecycle roles.

The builder shall not execute any deployment logic. Its sole responsibility is to construct and initialize runtime metadata.

---

# Business Value

Provides a standardized runtime representation of every artifact.

Removes direct dependency between lifecycle roles and the Manifest.

Ensures every downstream role operates on the same contract.

Provides the foundation for:

- Parallel execution
- Reporting
- Checkpointing
- Resume
- Notifications

# Responsibilities

## Read Manifest

Read every artifact defined in the manifest.

Do not modify the manifest.

---

## Read Resolved Environment

Consume the resolved environment produced by Environment Resolver.

Examples

- environment
- region
- report root
- framework root
- notification recipients

---

## Create Runtime Metadata

Create a new Artifact Metadata object.

Populate

User supplied fields

- artifactId
- releaseVersion
- productionVersion
- runtimeUser
- deployProduction

Resolved fields

- environment
- region

Framework generated fields

- workspace
- reportDirectory
- logDirectory
- checkpointDirectory

Runtime state

- status=PENDING
- currentStage=INITIALIZED
- retryCount=0
- startTime=null
- endTime=null

---

## Generate Runtime Paths

Generate deterministic paths.

Examples

workspace

/apps/regression/work/<run-id>/<artifact>

reports

/apps/regression/reports/<run-id>/<artifact>

logs

/apps/regression/logs/<run-id>/<artifact>

checkpoint

/apps/regression/checkpoints/<run-id>/<artifact>

---

## Initialize Runtime State

Every artifact shall start with

status

PENDING

currentStage

INITIALIZED

No lifecycle role shall initialize runtime state.

---

## Validate Generated Metadata

Verify

workspace generated

report path generated

environment populated

runtime user populated

artifact id populated

Fail immediately if mandatory runtime metadata cannot be generated.
